dalias
======
Dynamic bash aliases (loosely based on [rbenv](https://github.com/rbenv/rbenv)).

Installation
------------
The simplest method is a git checkout:
```sh
$ git checkout https://github.com/t3hpr1m3/dalias.git ~/.dalias
$
```
